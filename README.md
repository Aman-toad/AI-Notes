# 📝 AI Notes

A modern and minimal notes app built with **Next.js**, **MongoDB**, **NextAuth.js** and **Tailwind CSS**.  
Create, edit, and organize your notes seamlessly with a clean UI and fast performance.

---

## 🚀 Features

- **Create, update, and delete notes**
- **Clean folder structure with Next.js App Router**
- **MongoDB database integration with Mongoose**
- **Tailwind CSS for modern responsive UI**
- **No empty notes allowed — keep your list tidy**
- **Fast refresh & no duplicate creation**
